Mac 上编译 CUDA+Tensorflow

  • MacOS High Sierra Version 10.13.6（17G6030)
  • NVIDIA Web Driver 387.10.10.10.40.124
  • CUDA Toolkit 10.0
  • cuDNN 7.5
  • NVIDIA Collective Communications Library (NCCL) 2.4.2-1
  • Python 3.6.8
  • Anaconda 2019.3
  • Xcode 9.4
  • Command Line Tool for Xcode 9.4

编译前准备

  • 1.禁用 System Integrity Protection（SIP）
  • 2.安装 Xcode 9.4
  • 3.安装 Anaconda 并安装 Python 3.6.8
  • 4.安装 NVIDIA Runtime
    • 1.安装 NVIDIA Web Driver
    • 2.安装 CUDA Toolkit
    • 3.安装 cuDNN
    • 4.安装 NVIDIA Collective Communications Library (NCCL)
  • 5.使用 Homebrew 安装库
    • 1.安装 coreutils
    • 2.安装 openmp
  • 6.安装 Jdk 8
  • 7.安装 Bazel

禁用 SIP

打开终端输入，关闭系统完整性保护

安装 Xcode 9.4

下载 Xcode9.4.xip
苹果开发者下载平台
Xcode 9.4
随后解压拷贝至 Applications, 打开终端运行将 Xcode 9.4 设置为默认

若你之前安装过其他版本的Xcode可能会安装匹配的Command Line Tool，需要更换其版本
Command Line Tool for 9.4

安装 Anaconda

在官网下载 Anaconda 并安装
Anaconda 下载页面
官网安装文档
Anaconda 安装向导
安装完成后，打开终端更换Python版本为3.6.8

随后，输入 Python 验证是否为3.6.8版本

安装 NVIDIA Runtime

  • 1.安装 NVIDIA Web Driver
  • 2.安装 CUDA Toolkit
  • 3.安装 cuDNN
  • 4.安装 NVIDIA Collective Communications Library (NCCL)

安装 Web Driver

选择与自己系统匹配的版本，下载下来并安装
下载地址

安装 CUDA Toolkit

在 archive 选择 CUDA Toolkit 10.0下载
CUDA Toolkit Archive
下载后安装，并配置变量到 .bash_profile

随后输入 nvcc -V 测试是否配置正确

检测 CUDA 是否能正常工作，切换目录到 CUDA Samples

若出现以下情况，请说明 Command Line Tool 版本不受支持，请更换版本

安装 cuDNN

下载 cuDNN 并解压到 CUDA 目录下，注意选择对应的 CUDA 版本
cuDNN 下载地址
下载后解压到 CUDA 目录

安装 NCCL

下载 NCCL
NCCL 下载地址
下载后解压，并配置软链接，注意替换成自己下载的版本

Homebrew 部分

安装 coreutils

安装 openmp

安装 Jdk

下载 Jdk 并根据向导安装 (解决Bazel安装问题，请下载JDK8)
Jdk 下载地址
验证 Jdk 是否安装成功

安装 Bazel

使用新版本的 Bazel 可能会导致编译失败，现在需要下载低版本（0.19.2）的 Bazel 来进行编译

Bazel 0.19.2

下载安装完成后配置环境变量

环境变量的配置根据 Bazel 安装的位置来进行确认，这里默认安装的是用户的主目录

激活环境变量

打开控制台输入

开始编译

  • 2.编译配置
  • 3.开始编译
    • 1.Bazel 编译
    • 2.PIP 包编译

下载 Tensorflow 源码

从 github 上下载 tensorflow的源码

默认使用 master 分支，如需要调整成其他分支，请如下配置（r1.14以上版本 Bazel 需要 0.24 以上如需编译请更换 Bazel 版本）

由于 Mac 不支持 align(sizeof(T)) 所以需要将他们移除

配置编译

打开控制台输入

根据提示进行配置，除了是否使用 CUDA Support 其他都默认写 No

其中 Additional compute capability significantly GPU计算能力需要到官网找到你的设备，然后填写数值或者安装 CUDA-Z查看

点击查看文档地址

开始编译

先使用 Bazel 进行编译

下图为开始编译

编译的时间需要根据不同的情况确定，这里大概花了1小时左右的时间，显示编译完成后，开始编译pip安装包

Bazel 编译完成后，开始编译PIP安装包

上图编译完成后会在

/tmp/tensorflow_pkg

目录下生成 whl 文件，随后使用pip安装即可
